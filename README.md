# ariawase

ariawase is an open-source distributed VBA libraries.
ariawase contains Class files, Module files and vbac (vbac.wsf).
vbac is not VBA compiler. Instead, this tiny script frees VBA code from binary files.

## contents

* vbac.wsf
* /src/ariawase.xlsm
	* Assert.bas
	* CdoMail.cls
	* Core.bas
	* Func.cls
	* IO.bas
	* MonkeyTest.cls
	* Resx.bas
	* Tuple2.cls
	* Util.bas
	* WinInet.bas
